# DTC Finder Pro

**Diagnostic Trouble Code Management Tool**  
Developed by Eng. Omar Younis — Microvast Inc.

---

## Download

Go to the **[Releases](https://github.com/OmarYounis01/DTC-Finder-Pro-Releases/releases/latest)** page to download the latest installer.

## Features

- Search DTCs by SPN, FMI, DTC Code, or Failure Name
- Colour-coded severity levels
- Cart system for collecting DTCs across projects
- Document generation (Word / PDF) with customisable field selection
- Pre-loaded DTC data for NDE, MVI Gen 1, MVI Gen 2, and MV-B/MV-C
- Database backup and restore
- Auto-update notifications

## System Requirements

- Windows 10 or later
- No additional dependencies required — everything is bundled in the installer

## Installation

1. Download `DTC_Finder_Pro_Setup.exe` from the [latest release](https://github.com/OmarYounis01/DTC-Finder-Pro-Releases/releases/latest)
2. Run the installer and follow the wizard
3. Launch DTC Finder Pro from the Start Menu or Desktop shortcut
